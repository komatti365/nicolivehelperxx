# New NicoLive Helper

ニコニコ生放送（リクエスト放送）をサポートするFirefoxアドオンです。
ステイン(@komatti365)が独自の改造を加えています。
動作が安定しない部分があるので原則は以下の本家バージョンを使ってください

# Link(本家)
- https://addons.mozilla.org/ja/firefox/addon/new-nicolive-helper/

## Discord の改行対応

Discord 投稿機能のテキスト入力欄を `textarea` 化し、複数行のメッセージを送信できるようになりました。
オプション画面で改行を入力するとそのまま Discord に反映されます。
デフォルト値には `\n` を含めることも可能です。
